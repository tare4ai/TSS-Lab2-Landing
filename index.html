Актуальність проєкту

p>У сучасному ритейлі утримання клієнтів є критичним. Система Bot-Loyalty автоматизує цей процес через Telegram, роблячи програму лояльності доступною без встановлення додаткових мобільних застосунків.

Мета дослідження

Розробка гнучкої та універсальної архітектури чат-бота для управління бонусними програмами лояльності малого та середнього бізнесу.

Основні завдання:

  • Дослідження методів ідентифікації користувачів у месенджерах. [cite: 107]
  • Проєктування бази даних для зберігання транзакцій та балів. [cite: 107]
  • Розробка серверної частини на Python (aiogram + SQLAlchemy). [cite: 107]
  • Створення модулів адміністрування на Go та Java. [cite: 107]

Очікувані результати

Створення програмного комплексу, що дозволяє бізнесу налаштувати власну систему лояльності за 15 хвилин, підвищуючи частоту покупок на 20-25%.
